
# SemiCode Analyzer

### ¿Que es?

- Es una calculadora para mostrar la configuración electrónica de un elemento químico en base a su numero atómico ademas de poder calcular que tipo de conductor se obtiene al dopar una con otro ya sea:
  - Intrínseco
  - Semiconductor P o N
- El proyecto fue realizado como una practica escolar para la BUAP.

### Instrucciones de creación:

1. Elaborar una aplicación en el lenguaje de su preferencia, que calcule la 				configuración electrónica de dos elementos en base a su número atómico.
2. Posteriormente, la aplicación determinara cuantos electrones de valencia y cuantos huecos tiene cada elemento en su ultimo nivel de energía.
3. Finalmente, realizara la combinación de los dos elementos y determinará si se produce un material aislante (intrínseco), tipo P, tipo N o ninguno de los anteriores.

**Opcional:**

4. Representar los diagramas de los átomos y sus enlaces como material N y P, mostrando los huecos o electrones libres.

5. Representar diagramas de energía (Bandas de valencia y de conducción).
